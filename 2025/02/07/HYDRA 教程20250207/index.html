双击打开软件后要求选择APP PACK位置

  • 选择下载好的 app pack位置

打开软件成功

  1. 我们只用到这两个设置

打开PBO2

我们先填入大概起始负压,就从什么负压开始测,填入对应的PPT EDC TDC 等按你需要的来

例如,设置后按保存按钮

起始负压比较省时间,不然要多跑几个小时

我们先来 金银核 - 20 其他核-30

点击SCAN 按钮

  • 确保只勾选第一项
  • 下面按钮导入之前pbo2哪里的设置,这就是快速导入 之前的设置,不用在这个界面重新设置,省点时间,重复测试有用
  • 点击确定开始测试
  • 测试过程会不断冻死屏 死机 黑屏， 请手动重启，重启后会自动继续测试，过程需要3-6小时

测试完成后如CCD#0 AMD CO: ( -25X -20X -47 -24X -40X -50 -49X -48X )

有些核-49x -50x ,负压太高l,可能 自己测yc 018 也有可能继续出错,我们采用抹平最高负压,利用此软件是寻找最差核心负压
修改一下
AMD CO: ( -25X -20X -37 -24X -39X -40 -40 -40 )
抹平最高为-40
这样怎么测试都不会报错了

注意事项:

  • 测试前先确保内存小参稳定,能过tm5测试等,

  • curve shaper max +13 high +3 确保得到最佳负压,这个要开,能修正高频稳定性—(这是针对9950x的,9800x3d因为频率只有5.45g,尝试 max +10 high +2,或者进一步缩小)

  • AMD 全核负压是采用单ccd内最差核心的负压值,所以上面测试结果基本就是全核采用core 2的负压值

  • 由于金银核心 本来就雕,默认电压低,所以负压值会偏低,但是金银核心-20 的负压 和差核心 -40 的负压 电压是差不多的,就是说最雕核 和最差核负压差距>10

  • 追求极限稳定的在上面抹平最高负压基础上再 +1 或者+2

关于内存:

  • 见大多数人上来就 6400 c28 ,但是你要自动能稳定6400 c28的u需很高的 soc电压1.3v 内存电压1.55v以上
  • 因为6400 即使fclk2133,瓶颈不在fclk,瓶颈在uclk ,基本上有一半的u不能稳 uclk 3200
  • 推荐中等水平超频选手上来就6200 2067

soc电压不足导致的uclk不稳是yc fft和tm5直接报错

  • 我的soc 1.15v fclk2200 FFTV4 报错
  • 切换为soc 1.2v ,FFTV4无问题了
  • 感觉vsoc1.2v下延迟也好看了,带宽也好看!!!这还是 core tuning auto下的延迟
  • FF14 能跑50774
  • YC8.5 FFTV3速度
  • 8.6 FFTV4

如果没有风扇,放宽TRFC-500+

TRRFI 除以2,40000以下,
如果不稳加大soc电压1.2v吧

  • 打开界面如下
  • 按0 -enter-1-enter-8 测试32g内存imc稳定性
  • 拉长显示界面,不然按2后,有些显示不出来

关于 curve shaper

  • 新手可能不了解

用途

  1. 动态调压与频率优化
    Curve Shaper 是 AMD 锐龙 9000 系列（Zen 5 架构）新增的超频功能，作为 Curve Optimizer 的扩展，提供 15 个细分调试点，覆盖不同频率段（如最低频、低频、中频、高频、最大频率）和温度条件（低温、中温、高温）。用户可通过调整电压偏移，精准控制 CPU 在不同负载下的电压与频率关系，避免低负载或低温时不必要的频率飙升，同时确保高负载性能不受影响2514。
  2. 解决低负载稳定性问题
    传统 Curve Optimizer 全核负压可能导致低负载死机，而 Curve Shaper 允许针对高频段（如游戏负载）单独优化，例如为高频高温段加压以增强稳定性，从而减少因电压不足导致的崩溃
  3. 与 Curve Optimizer 协同工作
    Curve Shaper 可与 Curve Optimizer 叠加使用，前者针对特定频率段优化，后者调整全局电压曲线，两者结合可实现更精细的性能与能效平衡1014。
  4. 通过主板 BIOS 设置
  • 进入 BIOS（开机时按 Del/F2/F10 等键），在 AMD Overclocking 或 Precision Boost Overdrive（PBO） 选项中找到 Curve Shaper 功能。
